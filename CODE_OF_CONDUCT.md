# Code of Conduct

## Our Pledge

This is a personal portfolio and learning project. While it is open source and
welcomes questions, suggestions, and constructive feedback, it is maintained by
a single person and is not a community-driven open source project in the
traditional sense.

All interactions in this repository (issues, pull requests, discussions) are
expected to be respectful, constructive, and professional.

## Our Standards

Examples of acceptable behaviour:

- Asking questions about the implementation, architecture, or tooling choices.
- Suggesting improvements via issues or pull requests with clear explanations.
- Pointing out bugs or inconsistencies respectfully.
- Being patient — this is a solo learning project maintained in spare time.

Examples of unacceptable behaviour:

- Disrespectful, dismissive, or harassing comments.
- Spam, self-promotion, or off-topic discussions.
- Demanding features, fixes, or responses on a specific timeline.

## Scope

This Code of Conduct applies to all interactions within this GitHub repository:
issues, pull requests, commit comments, and discussions.

## Enforcement

As the sole maintainer, I reserve the right to close issues, reject pull
requests, and block users who engage in unacceptable behaviour, without
obligation to provide a detailed explanation.
